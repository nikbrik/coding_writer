# Product Bugs

Этот каталог хранит product/doc bugs, которые были выявлены в ходе работы
agent harness или ручных сценариев.

Правило: если root cause находится в приложении, документации продукта,
архитектуре, UX, lifecycle, verification или persistence, сначала заводится
bug здесь. Harness learning или workaround может быть только вторичным
контекстом, но не заменой product fix.
